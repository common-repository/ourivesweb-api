O Ourives Web é um produto com a qualidade da PONTO 25 – informática lda.

== Description ==
Com quase 20 anos de experiência, a PONTO 25 é uma empresa especializada no desenvolvimento de soluções informáticas.

Através da utilização de tecnologias de informação interativas, a PONTO 25 disponibiliza um vasto leque de soluções de gestão desktop e na cloud para diversas áreas do mercado.
Sedeada em Braga, a PONTO 25 está presente em todo o país, Espanha, Moçambique, Angola, Brasil, Polónia e França.


Certificado pela Autoridade Tributária sob o número 1541, o OurivesWeb está sempre de acordo com a lei em vigor.


== Através do plugin é possível:  ==
* Sincronizar artigos e stocks entre as duas plataformas
* Emissão automática ou manual de documentos
* Seleccionar a secção da empresa.
* Criação automática de clientes e artigos
* Aceder aos documentos emitidos sem sair do WordPress

Todo o suporte técnico e comercial dado aos utilizadores do plugin é prestado pela equipa de Apoio a Clientes do OurivesWeb.

== Frequently Asked Questions ==


= Tenho dúvidas ou sugestões, quem posso contactar? =
Para qualquer duvida ou sugestão pode entrar em contacto connosco através do email apoiotecnico@ponto25.pt.

= Os documentos estão a ser emitidos sem contribuinte =
Por defeito, o WooCommerce não tem um campo de contribuinte, desta forma a Ponto25 disponibliza outro plugin que faz a integração do VAT europeu (que inlcui o Nif Portugues).

Este plugin cria um novo campo, como por exemplo `_billing_vat`,em que é necessário que o utilizador escolha nas configurações do plugin.

Depois de ter um plugin instalado, basta seleccionar nas configurações do plugin OurivesWeb qual o `Contribuinte do cliente`.


== Installation ==
Este plugin pode ser instalado através de FTP ou utilizando o instalador de plugins do WordPress.

Via FTP
1. Upload dos ficheiros do plugin para o diretório `/wp-content/plugins/ourivesweb-api`
2. Ativar o plugin através da opção `Plugins` visível no WordPress
